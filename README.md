# CVD Deep Learning Project

## About This Project

This project was completed as part of a Research Experience for Undergraduates (REU) in Data Science and Analytics at North Carolina A&T State University in Summer 2024.

**The problem:** Cardiovascular disease is one of the leading causes of death globally. Early prediction of heart failure events from patient data can meaningfully improve clinical outcomes, but the relationships between clinical features and mortality risk are complex and nonlinear.

**The approach:** Built and compared CNN and DNN architectures using TensorFlow and Keras to predict heart failure events from structured patient data. Applied SHAP (SHapley Additive exPlanations) to interpret model outputs and surface the most clinically meaningful predictors across patient cohorts.

**Key work:**
- Designed and trained CNN and DNN models for binary classification of heart failure events
- Applied data normalization and preprocessing pipelines to structured clinical data
- Used SHAP values to interpret model predictions and identify high-impact features
- Evaluated models using standard classification metrics (AUC, F1, precision/recall)
- Visualized results and model performance across patient cohorts

**Tech stack:** Python · TensorFlow · Keras · SHAP · NumPy · Pandas · Matplotlib · Jupyter
